
# Task Form Fields

This directory contains reusable form field components for task forms.

## Components:

- `TaskTitleField.tsx` - Title input field
- `TaskDescriptionField.tsx` - Description textarea field
- `TaskProjectField.tsx` - Project selection dropdown
- `TaskAssigneeField.tsx` - Assignee selection dropdown
- `TaskDateField.tsx` - Date picker field (used for both start and due dates)
- `TaskStatusField.tsx` - Status selection dropdown
- `TaskFormActions.tsx` - Form action buttons (Submit/Cancel)

These components are designed to be used with react-hook-form and the taskSchema defined in `../schemas/taskSchema.ts`.
