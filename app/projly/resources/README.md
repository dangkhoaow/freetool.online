# Resources Module

This folder contains the main page and supporting components for managing resources in Projly.

## Main Page
- `page.tsx`: The entry point for the /projly/resources route. Displays a table of resources, filtering options, and creation dialogs.

## Supporting Components
- All UI and logic for resource creation, editing, filtering, and display are in `/app/projly/components/resources/`.

## Logging & Error Handling
- Follows the Projly convention of detailed logging after variable changes and robust error handling.
