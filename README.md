# FreeTool.Online

A collection of free online tools for everyday use, starting with an AI-powered HEIC converter.

## Features

- **HEIC Converter**: Convert HEIC images to JPG, PNG, WEBP, and PDF with AI optimization
- Fast, secure, and easy to use with batch conversion support
- All processing happens in your browser - your files never leave your device

## Technologies

- Next.js
- React
- Tailwind CSS
- TypeScript

## Getting Started

```bash
# Install dependencies
npm install
# or
yarn install
# or
pnpm install

# Run the development server
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
